# Tesla Model 3 — Beginner Guide (2021 facelift)

Day-1 guide for new owners. Covers charging, controls, Autopilot, and maintenance.

## 1. Lading (uten hjemmelading)

| Scenario | Strategi |
|:---------|:---------|
| **Supercharger** (langtur / batteri ~20%) | Bare plugg inn — Tesla trekker betaling. Naviger til stasjonen i bilens navigasjon for å forvarme batteriet |
| **Butikksenter** (Circle K, Recharge, Mer) | Bruk app (Circle K, Mer, Recharge) eller ladebrikke |
| **Offentlig gateparkering** | App eller ladebrikke |

### Viktige lade-regler

| Regel | Hvorfor |
|:------|:--------|
| Hold 20-80% til daglig | Batteriet varer lengst |
| Lad til 90-100% kun før langtur | Skal du kjøre KRS-Oslo — lad til 100% før avreise |
| Supercharger: stopp ved 60-80% | Etter 80% synker ladehastigheten kraftig |
| Sett navigasjon til Supercharger | Bilen forvarmer batteriet automatisk |
| Flytt bilen når lading er ferdig | Unngå idle fee (7 kr/min kun ved >50% stasjonskapasitet) |
| Tesla-appen viser alt | Ladestatus, pris, estimert tid |

### Tesla Supercharger priser (Norge 2025/2026)

- **Standard**: ~3,50-4,50 kr/kWh
- **Med medlemskap** (99 kr/mnd): ~2,50-3,00 kr/kWh
- **Sammenlignet**: Supercharger er 20-50% billigere enn Ionity, Recharge, Circle K, Mer

### KRS-Oslo (330 km) oppskrift

1. Lad til 100% kvelden før (Supercharger nærme deg)
2. Naviger til Oslo i bilen
3. Bilen foreslår Supercharger-stopp selv
4. Stopp 1 gang: Porsgrunn/Larvik Supercharger (10-15 min)
5. Fortsett til Oslo med ~20-30% igjen

## 2. Første timer i bilen

### Skjermen

| Område | Funksjon |
|:-------|:---------|
| Venstre (bil-ikon) | Låser, lading, lys, frunk/bakluke |
| Midten | Kart og navigasjon |
| Høyre | Musikk, media, lyd, kamera |
| Sveip ned | Hurtiginnstillinger (viskere, dashcam, varme, etc.) |
| Sveip opp | Klima, setevarme, rattvarme |

### Starte og stoppe

| Handling | Resultat |
|:---------|:---------|
| Sett deg inn + trykk brems | Bilen våkner (ingen knapp) |
| Spake: D / R | Kjør eller rygg |
| Spake: P-knapp på enden | Parker |
| Gå ut med telefonen i lomma | Bilen låser seg automatisk |

### Spaker

| Spake | Bevegelse | Funksjon |
|:------|:-----------|:---------|
| Høyre (høyre side) | Opp/ned | Cruise-hastighet |
| | Inn/ut (1x) | Aktiver trafikkbevisst cruise |
| | Inn/ut (2x) | Aktiver Autostyring |
| Venstre | Opp/ned | Blinklys |
| | Inn | Tåkelys/horn |
| | Hel opp/ned | D (kjør) / R (rygg) |
| | Endeknapp | P (parkering) |

## 3. Autopilot 101

| Antall trekk på spaken | aktivert |
|:----------------------|:---------|
| **1 gang** (mot deg) | Trafikkbevisst cruise — holder fart + avstand |
| **2 ganger** (mot deg) | Autostyring — bilen styrer seg selv i feltet |

### Autopilot-regler

| ✅ Du må | ❌ Du må IKKE |
|:---------|:-------------|
| Ha hendene på rattet | Slippe rattet (bilen piper) |
| Følge med på veien | Se på film mens du kjører |
| Være klar til å overta | Stol blindt på autopilot |

Autopilot er som en lærling: flink på motorvei, men se opp i svinger og kryss.

## 5. Tesla-appen (gratis)

| Funksjon | Beskrivelse |
|:---------|:------------|
| Låse/låse opp | Alltid, uansett avstand |
| Forvarme | 5 min før avreise — isfrie ruter, varm kupé |
| Se ladestatus | Resttid, pris, % |
| Finn bilen | GPS-sporing |
| Sentry Mode på/av | Start kameraopptak eksternt |
| Send adresse til bil | Skriv på telefonen → åpner seg i bilens navigasjon |

## 6. Tips for nye Tesla-eiere

1. **Bruk navigasjonen** — den lærer kjøremønsteret ditt og justerer rekkeviddeestimatet
2. **Frunk (panser)** åpnes kun fra app eller skjerm — ingen fysisk knapp
3. **Bakluken** kan fotåpnes: stå bak, sving foten fort under støtfangeren
4. **Joe Mode** (Innstillinger → Sikkerhet) — demper varsellyder 50%
5. **Pin to Drive** — aktiver for tyverisikring (Innstillinger)
6. **Nøkkelkort i lommeboka** — backup hvis telefonen dør
7. **Vinterrekkevidde** — forvent 60-80% av WLTP ved -10°C, ca 30% økt forbruk
8. **Vindusviskere** — knapp på enden av blinklysspaken (auto-modus er OK)
9. **Sentry Mode trekker strøm** (~1-2%/dag) — skru av hjemme om du ikke trenger det
10. **Servicemeny** — bil-ikon → Service → skriv «SERVICE» for skjulte tester

## 7. 2021-facelift spesifikke fordeler

Sammenlignet med 2019-2020 Model 3:
- Varmepumpe (30% bedre vinterrekkevidde)
- Doble vinduer foran (støysvakere)
- Elektrisk bakluke
- Trådløs lading på midtkonsollen
- Rattvarme
- Setevarme bak
- Sorte vinduslister (sportsligere)
- MCU2 Intel Atom-prosessor (raskere skjerm)

## 8. Vanlige spørsmål

| Spørsmål | Svar |
|:---------|:-----|
| Trenger jeg abonnement? | Nei. Basic Autopilot, navigasjon, app-kontroll er gratis. Bare Premium Connectivity (129 kr/mnd) er valgfritt |
| Hvor ofte må jeg lade på Supercharger? | Uten hjemmelading: ~1 gang/uke ved 1.500 km/mnd |
| Hva koster KRS-Oslo tur/retur? | Supercharger: ~350-450 kr. Bensinbil: ~660 kr |
| Kan jeg vaske bilen i bilvask? | Ja — men bruk «Free Roll»-modus (berøringsskjerm) i vaskehallen |
| Må jeg ha service? | Tesla anbefaler ingen fast service. Skift kupefilter og sjekk dekktrykk |
